⚡ 工作流程

三步开始传输

无需数据线，无需安装客户端，扫码即用

01

启动服务

在电脑端运行程序，点击「启动服务」按钮，程序自动获取本机 IP 并开启 HTTP 服务。

02

扫码访问

手机扫描程序生成的二维码，或在浏览器输入显示的访问地址，即可打开文件管理器。

03

传输文件

在手机端浏览目录、下载文件到本地、上传手机文件到电脑，流式传输无大小限制。

✨ 功能特性

为效率而生

精心设计的每一项功能，让局域网文件传输变得优雅高效

仿 Win11 界面

侧边栏导航 + 面包屑 + 工具栏，响应式设计完美适配手机、平板与桌面浏览器。

文件管理

浏览目录、下载文件、上传文件、新建文件夹、删除文件/文件夹，功能完整。

二维码扫码

自动生成访问地址二维码，手机扫码即用，无需手动输入冗长的 IP 地址。

快速访问

桌面、文档、下载、图片等常用目录一键直达，自动列出所有磁盘驱动器。

系统托盘

最小化到托盘后台运行，右键菜单快速退出或重启服务，不打扰日常工作。

配置热更新

修改端口、IP 等配置后自动生效，支持开机自启动，省去重复设置的烦恼。

路径安全防护

路径规范化 + 穿越段检查，防止目录遍历攻击，保障文件系统安全。

大文件支持

流式上传与下载，无文件大小限制，传输大体积视频、压缩包同样流畅。

🚀 快速开始

几行命令，即刻运行

确保已安装 .NET 10 SDK 与 Windows 系统，克隆仓库后即可构建运行

PowerShell 
# 克隆并进入项目
cd WebExplorer

# 构建项目
dotnet build

# 运行服务
dotnet run --project WebExplorer
1 启动程序，主窗口显示本机 IP 和端口
2 点击「启动服务」按钮
3 手机扫码或浏览器访问显示的地址
4 在手机端浏览、上传、下载文件
